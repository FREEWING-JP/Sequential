Vedere immagini grandi

Le immagini grandi possono essere rimpicciolite per entrare nello schermo o le si può scorrere in diversi modi.

Per scorrere immagini grandi:

  • Trascina con il mouse.
  • Usa i tasti di navigazione (frecce, Home, End, Page Up, e Page Dn).
  • Premi spazio, V, B o C.
  • Premi 0 o invio (solo sul tastierino numerico).
  • Premi 1-9 (solo sul tastierino numerico) per avanzare di una schermata nella direzione appropriata.

Premere spazio o 0 provoca uno scorrimento attraverso tutta l'immagine se possibile, o altrimenti fino alla riga successiva. Premere V o invio provoca uno scorrimento lungo tutta l'immagine se possibile, o altrimenti fino alla colonna successiva.

Tenere premuto Alt inverte la direzione di uno di questi tasti. B e C sono equivalenti a Alt-spazio e Alt-V, rispettivamente.

Non vengono visualizzate le barre di scorrimento.
