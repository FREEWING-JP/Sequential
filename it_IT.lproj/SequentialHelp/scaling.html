Modalità di scala

Modalità Descrizione
Dimensioni reali Mostra le immagini senza modificarne le dimensioni.
Dimensioni reali con DPI Ridimensiona le immagini secondo la loro risoluzione dichiarata.
Ridimensiona automaticamente Scales images to be as large as possible while minimizing unnecessary scrolling.
Adatta alla finestra/schermo Ridimensiona le immagini per evitare qualunque scorrimento.

Per il ridimensionamento automatico e per l'adattamento alla finestra o allo schermo puoi scegliere se le immagini possono essere ingrandite, rimpicciolite o entrambe le cose (non limitare). Le immagini indipendenti dalla risoluzione (come i PDF) vengono sempre ridimensionate senza limiti.

Puoi anche scegliere se applicare un fattore di scala costante.
