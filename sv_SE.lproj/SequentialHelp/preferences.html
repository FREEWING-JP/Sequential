Inställningar

Inställning Beskrivning
Antialias vid uppskalning Mjuka upp bilder när de visas som större än vad de egentligen är. Bilder som visas nedskalade är alltid uppmjukade.
Runda bildhörn Rund bildernas hörn svagt.
Anpassa fönster till bild vid start Gör automatiskt fönstret lagom stort när den första bilden laddas.
Enbart för enstaka bilder Anpassa fönstret enbart för dokument som består av en enda bild. Andra dokumentfönster skalas inte om.
Bakgrund Färgen och mönstret som visas runt bilder.
Helskärm på Vilken skärm som ska användas i helskärmsläge.
Max mappdjup Hur djupt ned i en mappstruktur bilder ska öppnas.
Huvudknappsklick Bestämmer vad som ska hända vid pekdonsklick. "Nästa sida" kan vara antingen vänster eller höger beroende på läsriktning.
Escape-tangenten Vad ett tryck på Escape ska göra.
