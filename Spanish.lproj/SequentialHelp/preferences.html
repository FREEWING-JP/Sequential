Preferencias

Ajuste Descripción
Usar suavizado al ampliar Suavizar las imágenes cuando son mostradas a un tamaño mayor del real. Las imágenes mostradas a un tamaño menor siempre son suavizadas.
Animar imágenes Reproducir GIFs animados.
Esquinas redondeadas Mostrar las esquinas de las imágenes ligeramente más romas.
Ventanas del tamaño de la imagen El tamaño de las ventanas será ajustado automáticamente cuando la primera imagen se cargue
Fondo El color o textura mostrado alrededor de las imágenes.
Pantalla completa El visor para ver las imágenes a pantalla completa.
Profundidad de carpeta: El número de niveles dentro de los directorios para abrir.
El clic principal va: Determina el comportamiento de los botones principal y secundario del ratón. "A la página siguiente" puede ser la izquierda o derecha según la dirección de lectura, pues "izquierda" o "derecha" pueden ser las páginas siguientes o anteriores.
