Préférences

Réglages Description
Antialiasing Smooth images when they are drawn at larger than actual size. Images drawn smaller are always smoothed.
Arrondir les coins des images Arrondis les coins supérieurs et inférieurs des images.
Fit windows to initial image size Automatically resize windows when the first image is loaded to fit.
Uniquement pour les images seules Only use “fit windows to initial image size” for documents that have exactly one image. Other document windows are not resized.
Fond La couleur de fond derrière les images.
Mode plein écran Utilise tout l'écran pour l'affichage des images.
Ouvrir dossier fils Nombre de dossier fils ou Sequential va chercher les images à l'intérieur du dossier sélectionné.
Action du clic principal Détermine l'action du clic principal et secondaire. “To next page” can be either left or right depending on the reading direction, whereas “left” and “right” can be either the next or previous page.
Touche espace Entrer en mode plein écran / Sortir du mode plein écran.
