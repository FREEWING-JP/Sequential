Preferences

General Description
Background The color and pattern displayed behind images.
Full screen on The monitor to use in full screen mode.
Dim other screens Black out other monitors while in full screen mode.
Resize to size of first image Automatically resize windows when the first image is loaded.
Only for single images Only use “fit windows to initial image size” for documents that have exactly one image. Other document windows are not resized.
Antialias when upscaling Smooth images when they are drawn at larger than actual size. Images drawn smaller are always smoothed.
Round image corners Slightly bevel the corners of images.
Escape key The behavior of the Escape key.
Navigation Description
Primary click goes… Determines the behavior of primary and secondary clicks. “To next page” can be either left or right depending on the reading direction, whereas “left” and “right” can be either the next or previous page.
When going backward… Which corner of the image to start at when navigating from an image to the image before it.
Maximum depth The number of levels down into folders to open.
Update Description
Check automatically Check for updates automatically once a week. When an update is available you can install it by choosing Sequential > Install Update…. Requires an internet connection.
