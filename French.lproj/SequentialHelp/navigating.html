Naviguer dans les images

Il est possible de naviguer de différentes manières.

Pour aller à l'image précédente ou suivante :

  • Choisissez Page > Suivante ou Page > Précédente.
  • Clic, clic secondaire ou option-clic n'importe où dans l'image.
  • Pressez + ou - (sur le pavé numérique uniquement).
  • Utilisez le défilement pour passer aux images suivantes.

Pour aller à une image spécifique:

  • Si vous souhaitez aller à la première ou dernière image, utilisez Première > page ou Dernière > page.
  • Choisissez l'image voulue dans le menu Page

Si vous essayer d'aller aavant la première page ou après la dernière, Sequential affichera une alerte.
