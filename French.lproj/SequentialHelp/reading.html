Sens de lecture

Il est possible de changer le sens de lecture des images dans le logiciel Sequantial. Le changement permet à la fois de changer le focus sur l'image à la l'ouverture ainsi que le sens de navigation dans entre les images.

Pour changer le sens de lecture :

  1. Sélectionnez le sens de lecture désiré dans le menu Tri.

Les comics occidentaux sont souvent présentés de gauche à droite. Les mangas peuvent être lus de gauche à droite (quand ils ont été retournés) mais aussi de droite à gauche (le sens original)
